## 1. 對淡江大橋的直接發聲（2026-04，最新）

### 1.1 NPP 官方聲明標題
> 「淡江大橋機車道實際騎乘空間僅 1.6 公尺 —— 時代力量：交通部要補的洞，不僅僅是車道，更是有問題的法規標準！」

### 1.2 聲明核心論據

**（1）數字事實**
- 2.5 公尺是「分隔島之間的總寬度」，**不是騎士實際行駛空間**
- 公路局為避開橋體結構物，兩側各預留路肩
- 扣除後 **騎士實際可用空間 1.6 公尺**
- 馬來西亞研究指出 **機車道至少 1.7 公尺** 方能安全行駛
- **1.6 公尺卡在安全紅線之下**

**（2）法規診斷**
- 問題根源：**《公路路線設計規範》定義模糊**
- 規範僅寫「分隔式機車道不得小於 2.5 公尺」
- 未界定此 2.5 公尺是「分隔物之間」還是「可供騎乘的淨空間」
- 導致公路局可將路肩、緩衝區算進 2.5 公尺

**（3）救援困境指出**
- 機車道與汽車道之間以實體分隔島完全阻隔
- **救護車無法從快車道橫向進入靠近傷患**

**（4）歷史前例對照**
- 引用 **重陽橋、大漢橋** 機車道狹窄導致事故的歷史
- 指出這是「建窄 → 事故 → 花錢拓寬」的反覆循環
- 意指淡江大橋正在重演同樣錯誤

### 1.3 NPP 具體訴求

1. **交通部長陳世凱正面回應**外界質疑
2. **提出機車道安全改善方案**
3. **啟動《公路路線設計規範》修正程序** — 明確界定 2.5 公尺的測量基準

### 1.4 訴求位置分析

這個聲明 **同時命中 Framing A（規範修法）與 Framing D（程序正當性）兩個戰略位置**：

| Framing | NPP 這次聲明是否觸及？ |
|---------|-----------------------|
| A 規範修法 | **✅ 明確觸及** — 要求啟動《公路路線設計規範》修正 |
| B 通橋後監督 | ❌ 未觸及 |
| C 230 億決策審計 | ❌ 未觸及 |
| D 程序正當性 | **⚠️ 間接觸及** — 未明寫「安全諮詢小組透明化」，但「規範定義模糊」暗示程序黑箱 |

**關鍵含意**：strategic_dialectic 兩次 run 的 framing 位置分析（尤其 A 被評為「與 TPP 距離過近」）需要重新驗證，因為 **NPP 已經踩進 A 的位置了**。

## 2. NPP 的機車路權歷史立場（2022-07）

2022 年台灣機車路權促進會舉辦「交通解嚴大遊行」，時代力量立委 **陳椒華**、**邱顯智** 到場力挺。

### 2.1 現場發言重點
- 批評交通政策「亂七八糟」「一群混蛋」
- 強調「**車種分流是不切實際的政策**」
- 主張讓機車、汽車、行人都擁有合理、安全的道路空間

### 2.2 NPP 推動的在地行動
- 在桃園、新北、台北交通局舉辦 **機車安全工作坊**
- 協調改善危險路口、優化路口交通號誌
- 考察二段式左轉的必要性
- 改善設計不良的待轉區

### 2.3 與機車路權促進會的關係
- **已建立合作默契**，2022 遊行共同站台
- 對照 `stakeholders.md`：機車路權促進會正在尋找政治盟友
- NPP 重啟這個合作線是 **現成組織槓桿**，不是空降

## 6. 給後續 agent 的關鍵 implication

- `stakeholders.md` 第 3 段「NPP 空白、機會點」應修正為：
  > 「NPP 已佔位於『規範修法』角度。戰略問題從『要不要發聲』變為『如何從黨部聲明升級為系統性戰略』」

### 6.2 strategic_dialectic 需要重跑的原因

先前兩次 run 基於以下錯誤前提：
- 前提 1：NPP 未發聲 → 可選擇任一 framing
- 前提 2：Framing A 與 TPP 距離近 → NPP 跟進風險高

**新事實下這兩個前提都需重估**：
- 新前提 1：NPP **已踩入 A 的位置**（要求規範修正）
- 新前提 2：既然已踩進 A，戰略問題是「從 A 擴展到 D」而非「在 A/D 之間選」

### 6.3 新的戰略命題（需 agent 驗證）

> **「NPP 不是要從 A/B/C/D 選一個，而是要從 `已發表的 A 聲明` 出發，縱向深化到 D（程序立法）+ 橫向輔助 C（決策審計）」**

這個命題比原 strategic_dialectic 的 D+C 建議更貼近現實 — 因為 A 聲明已經出去了，否認或轉向 A 的成本比疊加 D+C 高。

### 6.4 下一步行動（建議）

1. **重跑 strategic_dialectic**，把本檔（`npp_positions.md`）加進 evidence pack
2. 預期新 run 的判斷變動：
   - 降低對 A 的保留態度（因為 NPP 已發過，不再是 hypothetical）
   - 提高對「A→D 升級」路徑的評估
   - 重新審視與 TPP 的差異化（NPP 的 A 聲明 **比 TPP 的「降編省道」更上位**，已經是規範層而非個案層）
