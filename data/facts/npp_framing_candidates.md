# NPP 切入 framing 候選（A/B/C/D）

Last updated: 2026-04-22
Status: 人工初擬版。Day 7 由 `strategic_dialectic` agent 獨立驗證或推翻。

---

> **設計原則：必須有 3 個以上互相對立的 framing，reporter 才能辯證。**
> 只給一個 framing 是 confirmation bias，面試方看得出來。

---

## Framing A — 「規範本身就是問題」

### 核心主張
淡江大橋機車道 2.5m 合規範 — 但規範本身是 40 年前制定的，
沒有反映當代交通現實。NPP 主張：**推動《公路路線設計規範》修法**，
把「用路人實際體感安全」、「機車族比例」納入規範。

### 為什麼是 NPP 能打
- NPP 的 DNA 是 **lawmaking** — 二次民主改革、制度改革
- 超越個案、超越藍白綠對立
- 規範修法是 4 年工作，不是 4 週議題 → 符合 NPP 長期性格

### 反證（面試會被問的）
- 「規範是工程技術問題，為什麼是政治？」
  → 反駁：規範背後的「安全諮詢小組」組成、議事、公開機制都是政治決定
- 「太抽象，選民沒感覺」
  → 部分承認，需要搭配具體事故敘事作為 landing point
- 「TPP 會不會搶先提修法？」
  → 會，所以 **動作要快 + 條文要具體**

### 操作細節
- 立委/議員質詢：要求公開《公路路線設計規範》的修訂流程
- 政策白皮書：草擬「重大工程用路人體感評估機制」
- 社群：用「40 年沒改的規範」一句話帶出來
- 聯盟：跟機車路權促進會綁定

## Framing B — 「通橋之後的生活政治」

### 核心主張
不糾纏機車道（讓 TPP 去打），而是打 **「淡江大橋通車只是第一天」** —
輕軌擠爆、八里配套、淡海炒作、北海岸承載、環境監測 —
整合式都市議題的長期監督。

### 為什麼是 NPP 能打
- 正面議程、不做「反對派」姿態
- 時間尺度 3–5 年，符合 NPP 組織重建期的布局
- 結合 NPP 的「世代居住正義」敘事
- 新北市府施政可以 **逐項追蹤**，有檢查點

### 反證
- 「沒有熱度，沒人 care」
  → 承認，但我們不追短期熱度，建立長期資料庫
- 「配套問題是地方政治，NPP 新北無組織」
  → 部分承認，但我們可以做監督平台而非選舉動員
- 「跟地方團體競爭」
  → 應是合作而非競爭

### 操作細節
- 建立「淡江大橋通車後監督儀表板」— 輕軌班次、事故數、淡海房價、空污
- 政策白皮書：重大工程通車後 6/12/24 個月檢討機制
- 社群：每月一篇「通車之後第 N 個月」追蹤文
- 聯盟：在地文史、環團、輕軌用戶

## Framing C — 「230 億的機會成本」

### 核心主張
230 億 = 流標 7 次 + 追加 90 億的代價。如果投在公共運輸，
淡海輕軌路網能擴張多少倍？公車快線能開幾條？
問的不是「該不該建橋」，是 **「這種決策模式合理嗎」**。

### 為什麼是 NPP 能打
- 世代運輸正義 + 程序監督 NPP 雙打
- 財政正義 — NPP 關心公共資源分配
- 有數據感，適合做視覺化

### 反證（最強）
- 「橋已經蓋好，這種『不如當初』的論述是事後諸葛」
  → 部分承認，需要轉化為「下一個重大工程怎麼不再發生」
- 「會被說反對建設」
  → 強烈風險，尤其在地民眾
- 「DPP 會反擊：NPP 當年在不在場？」
  → 需考察 NPP 2015–2020 立院期間對淡江大橋的發言紀錄

### 操作細節
- 政策研究報告：230 億的 alternative use case 量化分析
- 不做社群熱推，做智庫型輸出
- 連結到下一個類似工程（淡北道路、其他橋樑）的事前監督

### 我的評估
**C 是最難打的**，但如果做得好，建立 NPP「決策審計」的品牌。
風險是被誤解為「反橋」，**執行成敗在敘事功力**。

## Framing D — 「程序正當性 — 為下一個橋做準備」

### 核心主張
淡江大橋的經驗教訓要如何制度化？下一個重大工程（淡北道路、
其他大橋）如何不要再重演流標 7 次、追加 90 億、安全諮詢
黑箱？NPP 主張：**立法建立重大工程的公民監督機制**。

### 為什麼是 NPP 能打
- NPP 歷史主張（監督、透明）完美連接
- 前瞻性 — 不打已蓋好的橋，打還沒蓋的橋
- 可以做成 **「重大工程監督條例」** 具體法案

### 反證
- 「法規已有很多，真的需要再立嗎？」
  → 現行採購法、工程會機制 **事後檢討** 為主，缺事前 + 過程監督
- 「公民監督會不會拖延工程？」
  → 設計上要平衡：資訊公開 ≠ 否決權
- 「誰來當『公民監督代表』？」
  → 可參考環評委員會模式，但改進其透明度

### 操作細節
- 法案研擬：「重大工程公民監督條例」草案
- 研究：國外類似制度（德國、荷蘭、新加坡）
- 社群：不搶熱度，做 policy wonk 路線
- 聯盟：都市計畫、工程、法律學界

## 我的初判：**B + D 組合最佳**

### 為什麼不選 A
- 與 TPP「機車路權」太近，容易被吞
- 技術規範修法門檻高，NPP 當前立院無席次難以推進

### 為什麼不選 C
- 敘事風險太高 — 反對已蓋好的建設是選票毒藥
- 在地民眾會直接厭惡

### 為什麼選 B
- 正面 + 長期 = 符合 NPP 目前重建期的布局
- 地面配套監督 = 讓在地居民受益（降低敵意）
- 有連續性議題 source（每月都有新進展）

### 為什麼選 D
- 法制化 = NPP 核心 DNA
- 前瞻性 = 不反對已做的、不跟 TPP 搶熱點
- 可以做具體法案，有 deliverable

### B + D 的化學反應
B 提供 **在地政治落地點**（淡江大橋後續監督） +
D 提供 **全國層級制度敘事**（重大工程監督條例）=
一個叫 **「重大工程後續監督行動方案」** 的完整 posture。
